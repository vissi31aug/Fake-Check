Natural Language Processing

Step 1- Cleaning Data

Machine learning data only works with numerical features so we have to convert text data into numerical columns. So we have to preprocess the text and that is called natural language processing. In-text preprocess we are cleaning our text by steaming, lemmatization, removing stopwords, removing special symbols and numbers, etc. After cleaning the data we have to feed this text data into a vectorizer which will convert this text data into numerical features.

Step 2- Lemmatization

Many words in natural language have different forms (e.g., "run," "runs," "running"), and treating each form as a separate feature can lead to sparsity and decreased accuracy. Lemmatization can address this issue by reducing all inflected forms of a word to its base form, so that they are treated as the same feature.

Step 3- TF-IDF Vectorizer

1. Term frequency = (Number of repetition of word in sentence / No of word in sentence )

2. Inverse Document Frequency = log(No.of sentence/No of sentence containing word)

3. Output Vector Matrix = TF * IDF
